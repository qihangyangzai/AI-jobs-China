# 中国AI职业冲击预测

> 123个职业 · 2024-2025年真实从业人数 · 交互式热力图

**[🔴 在线体验 →](https://liyizhouai.github.io/AI-jobs-China/)**

---

## 这是什么

一个可视化工具，把中国123个主要职业按 **AI替代风险** 展示成嵌套热力图。

- 面积 = 真实从业人数（来源：教育部、卫健委、国家统计局等2024年官方数据）
- 颜色 = AI暴露程度（🟢 低风险 → 🟡 中风险 → 🔴 高风险）
- 按行业聚类，点击展开行业查看细分职业
- 特别标注4类AI时代新职业（金色脉冲动效）：一人公司操盘手、知识付费从业者、智能体架构师、AI提效顾问

## 如何使用

**「按行业」视图（默认）**
- 鼠标悬停 → 查看职业简介
- 单击行业区块 → 平滑展开该行业所有细分职业
- 点击左上角「全部行业」→ 返回总览

**「全部职业」视图**
- 所有123个职业按从业人数等比平铺
